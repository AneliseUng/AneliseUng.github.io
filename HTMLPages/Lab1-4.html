Common HTML tags

These are the most commonly used HTML tags and a description of each.

Block elements

Block elements take up their own line and the full width of that page.

body

gives documents body

head

Info about the document

HTML

Defines root of html document

div

Defines a section

section

Defines a section

header

Defines header

main

specifies main content

article

Defines article

nav

Navigation links

footer

Defines a footer for a document or section

p

paragraph

aside

Defines content aside from the page content

li

defines list term

Inline elements

Inline elements only take up as much space as they need to: they do not start new lines.

a

defines a hyperlink

strong

Important text

em

emphasized text

img

Defins an image

span

Defines a section in a document
